# guía rápida para abrir y usar el proyecto en VS Code

## Opción 1: crear la carpeta desde el explorador de Windows
1. Ve a la ubicación donde quieras guardar tu TFM.
2. Crea una carpeta llamada `TFM-SGP-Insight`.
3. Descomprime allí el contenido de esta plantilla.
4. Abre VS Code.
5. Ve a **File > Open Folder**.
6. Selecciona la carpeta `TFM-SGP-Insight`.

## Opción 2: crear la carpeta desde VS Code
1. Abre VS Code.
2. Ve a **File > Open Folder**.
3. Crea una carpeta nueva con el nombre `TFM-SGP-Insight`.
4. Abre esa carpeta.
5. Copia dentro la estructura de esta plantilla.

## Recomendación inicial en VS Code
Una vez abras la carpeta:
- revisa `AGENT.md`
- abre `context/00_resumen_proyecto.md`
- abre `context/05_conexion_sqlserver.md`
- completa `context/02_diccionario_tablas.md` con tus tablas reales

## Qué hacer después
1. Crear el entorno virtual de Python.
2. Instalar dependencias.
3. Configurar el archivo `.env`.
4. Empezar a guardar scripts SQL en `sql/`.
5. Guardar ETL y validaciones en `python/`.
